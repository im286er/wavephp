session做了自定义处理，可以存到文件，也可以存到缓存，也可以存到数据库

配置：

```php
'session'=>array(
    'driver'            => 'db',
    'timeout'           => 86400
)
```

driver可以填：
1. file
2. db
3. memcache
4. memcached
5. redis
